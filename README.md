# Programming-Workspace

A collection of small example programs and exercises, organized by language.

## Folders

- `C/` — C language projects and exercises.
- `Html/` — HTML and frontend examples (HTML, CSS, JavaScript).
- `Java/` — Java projects and exercises.
- `Python/` — Python scripts and projects.

## How to use

1. Browse the folders above to find examples and exercises.
2. Open a folder and follow any language-specific README or instructions in that folder.
3. To add new code, create a new directory under the appropriate language and open a pull request.

Happy coding!
